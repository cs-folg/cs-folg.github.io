Java Project

In my third semester at UCF, I was tasked with creating a Java project in my Object Oriented Programming class. The project was to create a system of menus that a user could interact with to modify a College database. Options included adding, removing, and viewing students, professors, and courses. The project lasted the course of the semester and was done with three other students. Using Discord, we coordinated timelines and divided tasks equally.
